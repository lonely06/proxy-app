## 仓库结构

- `rules/`：共享直连 / 代理规则
- `profiles/`：各客户端配置
- `sing-box/`：sing-box 独立规则示例

## 主要使用环境

- OpenWRT
  - OpenClash
  - Sing-box / homeproxy
- PC
  - mihomo 内核客户端：Clash Verge Rev、FlClash
- iOS
  - Shadowrocket
  - Loon

## 规则来源

- `rules/`：仓库自维护直连 / 代理列表
- OpenWRT / Sing-box

- iOS 各类代理软件
